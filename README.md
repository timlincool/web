web
===

c# webform


使用C#建置webform網頁 內容為活動申請表



需連接資料庫才可使用網站

需根據本身環境修改Web.config下 testhwDBConnectionString字串

按下新增刪除後才解除其他欄位不可編輯的狀態

DB 帳號test 密碼test123

環境 MSSQL2012 , VS2013
